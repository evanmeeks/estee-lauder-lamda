# ELC Netlify CI Demo

## Getting Started

### Installation

```
nvm install v14.21.1 && nvm use v14.21.1 && yarn
netlify init
```

### Development Server
```
netlify dev
```

### Production Deploy

```
netlify deploy build production
```
